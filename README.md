# 📚 Mi Repositorio de Aprendizaje y Ejercicios

¡Bienvenido! Este repositorio funciona como mi cuaderno de notas digital y laboratorio de pruebas. Aquí almaceno cientos de ejercicios prácticos, retos de código, scripts sueltos y diagramas que voy realizando en mi día a día para consolidar conceptos de programación, bases de datos y arquitectura.

El objetivo es mantener un histórico de mi evolución técnica y tener código de referencia rápido al que acudir cuando lo necesite.

---

## 📂 Estructura del Repositorio

Para no perderme entre tantos ejercicios, el repositorio se organiza por áreas temáticas mediante carpetas independientes:

* **📁 `mis-diagramas/`** - Modelado técnico de procesos y diseños de arquitectura usando Draw.io.
* **📁 `ejercicios-sql/`** - *(Próximamente)* Consultas, optimización de queries y scripts de bases de datos relacionales.
* **📁 `estructuras-datos/`** - *(Próximamente)* Prácticas de lógica, algoritmos y resolución de problemas.
* **📁 `apuntes-teoricos/`** - *(Próximamente)* Hojas de trucos (cheat sheets) y resúmenes de tecnologías en Markdown.
